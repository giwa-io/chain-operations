# Set Init Bond for Dispute Game - GIWA Sepolia

## Description
This operation sets the initial bond for the dispute game.
When an OP Stack chain is first deployed, the dispute game's initial bond value is set to zero by default.
So, we update the value by this operation.

This task consisted of `DGF.setInitBond`.
